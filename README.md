# Todo App

React + TypeScript로 만든 Todo 애플리케이션입니다.

## 기능

- Todo 추가/삭제
- Todo 완료 처리
- 검색 기능
- 정렬 기능
- Todo 수정

## 기술 스택

- React 19
- TypeScript
- Vite
- CSS

## 실행 방법

```
npm install
npm run dev
```
